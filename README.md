## About

전주대학교 졸업 요건 관리 시스템 캡스톤 프로젝트

Flask + HTML/CSS/JS 기반으로 개발되었습니다.

## 프로젝트 구조

```
capstone/
├── backend/      # Flask 백엔드
├── frontend/     # HTML, CSS, JS 프론트엔드
└── docs/         # 프로젝트 문서
```

## 실행 방법

1. 가상환경 생성 및 활성화
   ```bash
   cd backend
   python -m venv venv
   venv\Scripts\activate   # Windows
   ```

2. 의존성 설치
   ```bash
   pip install -r requirements.txt
   ```

3. 서버 실행
   ```bash
   python run.py
   ```

4. 브라우저에서 http://localhost:5000 접속
